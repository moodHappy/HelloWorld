# Survivors are 'nervous and sceptical' about partial Epstein file release

> 📅 2025/12/21 10:43:46

- 概要与内容要点：按美国国会要求，美国司法部公布与已故性侵者Jeffrey Epstein相关的成千上万页材料，已发布的档案中包含照片、证词摘录和涉高知名度人物的材料等，但大量内容被涂黑，部分材料尚未公开。DOJ 表示被红acted 的信息主要涉及受害者隐私、儿童性虐待与暴力描写、可能影响正在进行的联邦调查，以及需要保密的机密资料等。司法部还强调不会删除政客的名字，除非他们本身是受害者；他们同时表示将按法定要求向国会提供关于被涂黑部分的日志，并称将遵循 Epstein 档案透明法案的规定。公开材料包括Ghislane Maxwell在 Downing Street 外的照片、Epstein 将一名14岁女孩介绍给特朗普等说法，以及多张涉及克林顿等名人的照片和 Epstein 的居所、出行等图像，但并非所有人名或图片都被指认有不当行为。

- 受害者与公众的反应：许多幸存者对公开内容感到紧张、怀疑与失望，担心剩余档案继续以相似方式被“分阶段”披露，使人难以获得完整信息并产生混乱的上下文。一位14岁时被虐待的幸存者 Marina Lacerda 表示希望尽快看到全部材料，并担心红acted 方式会让部分信息继续被隐藏。幸存者 Liz Stein 认为 DOJ 的做法与“爱泼斯坦档案透明法”相悖，担忧慢速、断章取义的披露难以让人了解真实证据。法律专家与议员也对大量红acted 表示质疑，担心这会助长阴谋论并要求对红acted 内容、披露范围和日志提供更清晰解释；部分民主与共和派议员呼吁进一步行动，甚至讨论弹劾或起诉等选项。

- 政府立场与未来展望：DOJ 表示将依法披露材料，并强调不会在涉及知名人物的名字上随意删除，除非与受害者相关；负责案件的官员也强调保护受害者隐私和正在进行调查的必要性，以及对照片中人物进行辨识的困难。同时，国会已要求在规定时间内提供被涂黑的日志，批评者则呼吁以更完整、具上下文的方式公开材料，避免误导公众。随着法案要求的持续执行，外界期待继续发布更多未删减的信息，但目前的高程度红acted 仍将继续成为透明度争议的核心议题。
